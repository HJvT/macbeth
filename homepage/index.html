Lentils - A simple and clean FICS client for OSX

What is FICS?

FICS is the Free Internet Chess Server. It is one of the oldest and largest internet chess servers.

View details »

What is Lentils?

Lentils provides a graphical interface for FICS. Since FICS is text-based, a program is needed to make FICS pretty. This is where Lentils comes in.

View details »

Where do I start?

Download Lentils and start playing as a guest on FICS. Later you can register for a FICS account, for free!
